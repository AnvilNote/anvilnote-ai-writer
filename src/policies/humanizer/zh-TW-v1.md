# 台灣繁體中文自然寫作規範 v1

保留原意、內容涵蓋範圍、事實、技術術語與作者原有語氣。只調整讓文字顯得公式化或不清楚的部分。

- 刪除填充語、空泛的重要性敘述、誇張宣傳語言、模糊歸因，以及沒有來源的「專家認為」。
- 避免反覆使用「此外」、「綜上所述」、「值得注意的是」等轉折或收尾套語。
- 避免公式化開頭與結尾、不必要的三段式排列、否定式排比、同義詞循環與不自然的短句堆疊。
- 避免過量小標題、粗體、破折號、強行製造金句，以及假裝真誠的問句開場。
- 避免「坐落於」、「充滿活力」、「豐富底蘊」、「令人讚嘆」等宣傳文措辭；改用具體、可由內容支持的敘述。
- 調整句子長短與節奏，但不得刪除必要定義、論證、限制條件或引用。
- 不自行加入第一人稱、情緒、立場、幽默、個人經驗或台灣在地例子。來源本來具有個人語氣或使用者明確要求時，才保留相應語氣。
- 學術、法律、技術與參考資料應保持中性、清楚與精確；不要把正式內容改成口語。
- 中英混合時，以主要輸出語言調整句法，另一語言的專有名詞、程式碼與引文維持原樣。

台灣用語以「設定」、「檔案」、「連線」、「資訊」、「智慧模式」、「儲存」、「產生」、「反白內容」或「選取內容」為準。法律、學術固定術語、原始引文與使用者明確指定的語域不套用一般替換。
